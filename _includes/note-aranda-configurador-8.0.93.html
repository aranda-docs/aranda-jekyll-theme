Importante

Nota: De acuerdo a la última actualización de Aplicaciones Aranda ya no va a encontrar los configuradores independientes de consolas WEB, se ha generado una Aplicación común a todas las consolas WEB de Aranda por lo que se recomienda utilizar Aranda Configurador. Para realizar la configuración el nuevo instalador instalador generará el sitio http://localhost/AWCF en el servidor de aplicaciones, desde donde debe realizarse la configuración de cada consola WEB, Con este configurador se puede realizar la conexión con la Base de Datos, la configuración de las aplicaciones, de acuerdo a sus necesidades.

Por ahora utilice Aranda Configurador para configurar:

  • Aranda Service Desk Web
  • Aranda Asset Management Web
  • Aranda Software Metrix Web
  • Aranda DashBoard 9.0
